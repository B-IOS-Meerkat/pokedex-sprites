# Pokédex Sprites

Sprites for Pokémon, sourced from [PokéAPI/sprites](https://github.com/PokeAPI/sprites).

## Variants

- `default/` - Front default sprite (96x96)
- `shiny/` - Front shiny sprite (96x96)
- `official-artwork/` - Official artwork (~1000x1000)
- `home/` - Pokémon HOME models (512x512)
- `home-shiny/` - Pokémon HOME shiny models (512x512)

## Access via CDN

Use jsDelivr for fast, free CDN access:

```
https://cdn.jsdelivr.net/gh/B-IOS-Meerkat/pokedex-sprites@main/<variant>/<id>.png
```

Example: `https://cdn.jsdelivr.net/gh/B-IOS-Meerkat/pokedex-sprites@main/official-artwork/25.png`
